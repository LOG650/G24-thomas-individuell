# Data Structures for Statistical Computing in Python

**Forfattere:** Wes McKinney
**År:** 2010
**Kilde/Journal:** Proceedings of the 9th Python in Science Conference (SciPy 2010), 56–61

---

## Sammendrag

Artikkelen presenterer pandas-biblioteket for Python, som tilbyr datastrukturer (Series, DataFrame, Panel) spesielt utviklet for statistisk beregning og dataanalyse. McKinney beskriver utfordringene med å jobbe med heterogene datasett i Python og hvordan pandas løser disse gjennom automatisk datajustering, håndtering av manglende verdier, grupperings- og pivotoperasjoner. Biblioteket er inspirert av R sitt data.frame-objekt, men med forbedringer for ytelse og fleksibilitet.

## Nøkkelord

`pandas` · `Python` · `datastrukturer` · `statistisk beregning` · `DataFrame`

## Relevans for oppgaven

Pandas er et sentralt verktøy for databehandling og analyse i Python-baserte AI/ML-prosjekter, og denne artikkelen gir det metodiske grunnlaget for programvareverktøyet som typisk brukes i implementering av AI-drevet lagerklassifisering.
