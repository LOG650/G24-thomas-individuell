# Scikit-learn: Machine Learning in Python

**Forfattere:** Fabian Pedregosa, Gaël Varoquaux, Alexandre Gramfort, Vincent Michel, Bertrand Thirion, Olivier Grisel, Mathieu Blondel, Peter Prettenhofer, Ron Weiss, Vincent Dubourg, Jake Vanderplas, Alexandre Passos, David Cournapeau, Matthieu Brucher, Matthieu Perrot, Édouard Duchesnay

## Sammendrag

Artikkelen presenterer scikit-learn, en Python-modul som integrerer et bredt utvalg av moderne maskinlæringsalgoritmer for overvåket og uovervåket læring. Biblioteket legger vekt på brukervennlighet, ytelse, dokumentasjon og API-konsistens. Scikit-learn er bygget på NumPy og SciPy, inkorporerer kompilert C++-kode fra LibSVM og LibLinear for effektivitet, og distribueres under BSD-lisens. Pakken tilbyr verktøy for klassifisering, regresjon, klyngeanalyse, dimensjonsreduksjon, modellvalg og forbehandling, og er designet for å gjøre maskinlæring tilgjengelig for ikke-spesialister.
